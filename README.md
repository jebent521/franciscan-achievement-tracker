# Franciscan Achievement Tracker

## Team Members
- Miguel Lock
- Evan Whitmer
- Jonah Ebent

## Project Summary
App where Franciscan students can track achievements earned in their time at Franciscan. Achievemnts range from simple to silly & elaborate.

## Technical Details

### Data Storage
- All JSON

### Server Stack
- **Database:** PostgreSQL
- **Backend:** Express.js

### Development
- **IDE:** VS Code
